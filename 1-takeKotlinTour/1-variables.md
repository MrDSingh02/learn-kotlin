## Hello World

Sample Program That prints '`Hello, world!`'.

```Kotlin
fun main() {
    println("Hello, world!")
    // Hello, world!
}
```

>In Kotlin:
>- `fun` is used to declare a function
>- the `main()` function is where your program starts from
>- the body of a function is written within curly braces
